Saving Project Data

There are a lot of parameters that need to be specified and made available to various function. We could just use global variables but that is sloppy. On the other hand, passing the parameter values a individual function arguments is a pain and error prone. A better option is to warp our parameters into a single object and pass that around, as in the following code:

param.G = 6.674e-11               # gravitational constant
param.M_earth = 5.972e24          # mass of the Earth [kg]
param.R_earth = 6371e3            # radius of the Earth [m]
param.mol = 0.029                 # molar mass of air
param.R = 8.314                   # gas constant
param.P0 = 101325                 # standard pressure (Pa), sea level 
param.T0 = 300                    # Florida temperature (K), sea level
param.g0 = 9.80665                # gravitational acceleration, sea level

# air density, sea level
param.rho0 = (param.mol*param.P0) / (param.R*param.T0)

Stored values can be accessed using the dot notation as in 

1
print("Acceleration due to gravity is", param.g0)

You can add extra parameters to param. The parameters listed above will be described in the following pages.

falcon.name = "Falcon 1e"

falcon.L = 27.4                      # rocket length, in meters
falcon.m_full = 46_760               # initial mass, in kg ("wet mass")
falcon.m_empty = 3_120               # mass when fuel is expended, in kg ("dry mass")
falcon.Isp = 221                     # specific impulse, in seconds

# calculated properties
