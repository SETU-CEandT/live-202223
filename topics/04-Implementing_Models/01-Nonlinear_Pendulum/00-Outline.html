Implementing Models

To date we have focused on core computational skills — root finding, integration, curve fitting; ode, etc — now we want to use those skills to study some models.

This week we will look at the damped non-linear pendulum.

Non-linear Pendulum

In this practical we will study a typical Physics problem using the technologies that we have covered to date. The problem, that I picked is, the behaviour of a nonlinear pendulum

This is a relatively simple problem but can have complicated behaviour. We will use this problem to practice using the numpy/scipy/matplotlib libraries.

In particular, we will

  • Compute the analytic and numerical solution for the linear problem.
  • Compare the numeric solution for the nonlinear problem to the solutions for the linear problem.
  • When given noisy data, determine estimates for the problem parameters (pendulum length and/or initial conditions).
  • Compute the period of the nonlinear problem using integration and special functions.
