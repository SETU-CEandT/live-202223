Function Implementation

Now that we have a basic implementation we want to wrap our code in a function so that we can use in within other code later

  • Create a text cell and enter a level 3 heading with title Function Implementation
  • Enter the following in a code cell.

2
3
4

6
def getLinearAnalytic(tValues, ic=np.array([1,0]), g=1, L=1):
    omega = np.sqrt(g/L)
    A = ic[0]
    B = ic[1]
    result =  A*np.cos(omega*tValues)+ B/omega*np.sin(omega*tValues)
    return result

Note:

  • I ordered arguments to function to allow giving (sane) default values. As a Physicist you might say that \(g=1\) is not exactly "sane". You are correct but when we often scale our problems so that we can work with easier values.

  • At this point in computational programming one would try to verify one's output by comparing this with related output - we will do this when generating the non-linear solution.

    plt.plot(tValues, linearAnalytic, label="$L=%s$" % L)

plt.ylabel('$\\theta(t)$')
