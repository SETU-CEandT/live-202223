Computing the Period

In this page we cover the relationship between the period and th maximum displacement.

Create a text cell and enter a level 2 heading with title NonLinear Pendulum (Period).

In the linear approximation the period is independent of the initial conditions and is given by

\[ T_0 = 2 \pi \sqrt{\frac{L}{g}} \]

However, for the non-linear problem the period depends on the initial position \(\theta(0)=\theta_0\) and can be represented by the following integral

\[ T = 4 \sqrt{\frac{L}{2g}} \int_0^{\theta_0} \frac{1}{\sqrt{\cos\theta - \cos\theta_0}} d\theta \]

This integral can be written in terms of elliptic integrals (see theory and scipy implementation) as

\[ T = 4 \sqrt{\frac{L}{2g}} F\left( \frac{\pi}{2}, \sin\left(\frac{\theta_0}{2}\right)\right) \] where \(F\) is the incomplete elliptic integral of the first kind.

We are going to evaluate both expressions, the first using quadrature (numerical integration) and the second using special functions in scipy.

Problem Type: Quadrature

Given a function \(f\) and an interval \([a,b]\) compute the definite integral

\[ \int_a^b f(x) \ dx \]

Process:

  • Import quad from scipy.integrate
  • Define function \(f\).
  • Call quad passing function, lower and upper limits, and any function additional arguments.

Application

  • Import quad from scipy.integrate

  • Import the required special function from scipy.special. Here we are importing the incomplete elliptic integral of the first kind ellipkinc.
