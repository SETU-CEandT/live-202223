General Programming Problems

max_of_two

Define a function, max_of_two, that takes two numbers as arguments and returns the largest of them. Use the if-else statement available in Python. (Note: Python has a max function built in, but writing it yourself is a good exercise.)

max_of_three

Define a function, max_of_three, that takes three numbers as arguments and returns the largest of them.

is_vowel

Write a function, is_vowel, that takes a character (i.e. a string of length 1) and returns True if it is a vowel, False otherwise.

translate

Write a function, translate, that will translate a text into 'Fake Swedish'. That is, double every consonant and place an occurrence of "o" in between. For example, translate("this is fun") should return the string "tothohisos isos fofunon".

add and multiply

Define functions add and multiply that adds and multiplies (respectively) all the numbers in a list of numbers. For example, add([1, 2, 3, 4]) should return 10 and multiply([1, 2, 3, 4]) should return 24.

reverse

Define a function, reverse that computes the reversal of a string. For example, reverse("I am testing") should return the string "gnitset ma I".

is_palindrome

Define a function is_palindrome that recognises palindromes (i.e., words that look the same written backwards). For example, is_palindrome("radar") should return True.

is_member_of

Write a function, is_member_of, that takes a value (i.e. a number, string, etc) x and a list of values a, and returns True if x is a member of a, and False otherwise. (Note that this is exactly what the in operator does, but for the sake of the exercise you should implement your own version using a loop.)

overlapping

Define a function overlapping() that takes two lists and returns True if they have at least one member in common, and False otherwise. You may use your is_member_of function, or the in operator.

max_in_list

The function max_of_two from question 1 and the function max_of_three from question 2 will only work for two and three numbers, respectively. But suppose we have a much larger number of numbers, or suppose we cannot tell in advance how many they are? Write a function max_in_list that takes a list of numbers and returns the largest value.

find_longest_word_in_list

Write a function, find_longest_word_in_list, that takes a list of words and returns the length of the longest word.

filter_short_words

Write a function, filter_short_words, that takes a list of words and an integer n, and returns the list of words in list that have at least n characters.

is_pangram

A pangram is a sentence that contains all the letters of the English alphabet at least once, for example: The quick brown fox jumps over the lazy dog. Write a function, is_pangram, check a sentence to see if it is a pangram or not.

Physics Programming Problems

time_to_fall

A ball is dropped from a tower of height $h$ with initial velocity zero. Write a function, time_to_fall, that accepts a single parameter (the initial height in metres) and returns the time the ball takes until it hits the ground, ignoring air resistance.

Use your function to calculate the time for a ball dropped from a 100 m high tower.

satellite_altitude

A satellite is to be launched into a circular orbit around the Earth so that it orbits the planet once every $T$ seconds.

The altitude~$h$ above the Earth's surface that the satellite must have is \[ h = \biggl( {GMT^2\over4\pi^2} \biggr)^{1/3} - R, \] where

  • \(G=6.67\times10^{-11}\,\textrm{m}^3\,\textrm{kg}^{-1}\,\textrm{s}^{-2}\) is Newton's gravitational constant,
  • \(M=5.97\times10^{24}\,\)kg is the mass of the Earth, and
  • \(R=6371\,\)km is its radius.

Write a function, satellite_altitude, that takes parameter, T, the desired satellite period, and then returns the altitude in meters.

  • Use your function to calculate the altitudes of satellites that orbit the Earth
  • once a day (so-called "geosynchronous" orbit),
  • once every 90 minutes, and

  • once every 45 minutes. What do you conclude from the last of these calculations?

  • Technically a geosynchronous satellite is one that orbits the Earth once per sidereal day, which is 23.93 hours, not 24 hours. Why is this? And how much difference will it make to the altitude of the satellite?
