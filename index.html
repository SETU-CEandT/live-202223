Module Overview and Introduction

Outline of module, list of resources, split of module between David's networking and my toolchains.

Intro to Python

This is the first of our python lecture/practicals. The aim of this lecture/practical is to introduce the main concepts in python and to get you coding small tasks as quickly as possible.

Scientific Computing using Python

In this weeks session we will introduce the trinity of numpy (high performance matrix library), scipy (scientific computation library) and matplotlib (2D and 3D plotting library). These python libraries have been instrumental in the current interest in using python for computational tasks.

Computational Problems involving Probability

Given that most of the problems that we will study this semester will have a stochastic component we will start with a review core concepts in probability and some of their applications.

Implementing Models

To date we have covered a number of basic computational tasks. We will now use these skills to implement two models — a non-linear pendulum and a rocket launch.
